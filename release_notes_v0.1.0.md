# v0.1.0

Initial public-safe release of the L Route Reflection Skill.

## Includes

- core `SKILL.md` workflow
- reusable templates
- example applications
- MIT license
- public-safe README

## Design Goal

Provide a reusable method for turning unstable-platform career narratives into:

- de-identified timelines
- route-comparison tables
- memoir drafts
- poster prompts
- risk/benefit audits

## Safety

This release is intentionally de-identified and suitable for public sharing.
