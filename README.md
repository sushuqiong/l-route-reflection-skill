# L Route Reflection Skill

This folder contains a real reusable skill package for turning unstable-platform career narratives into privacy-safe reflective outputs.

## Files

- `SKILL.md`
  Main skill instructions.
- `templates.md`
  Ready-to-use structures for timelines, route comparisons, audits, and essay prompts.
- `examples.md`
  Example applications.

## Best Use Cases

- memoir drafting
- career reflection
- unstable institution debriefing
- transformation of raw chat logs into essays, tables, and poster prompts

## Safety Default

Use aliases, remove searchable combinations, and separate effort from luck.
